# fastvue
快速构建vue的后台管理系统框架
## 项目启动步骤
### 下载项目git clone git@github.com:YesIsLast/fastvue.git
### 安装项目依赖 npm install
### 运行项目 npm run serve

## 项目组成
    vue 2.6+
    vue-router
    vuex
    eslint
    vue-progressbar VUE顶部加载轻量级动画https://github.com/hilongjw/vue-progressbar
